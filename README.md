# Ukkusuri Vamsi — Developer Portfolio

Modern dark-themed developer portfolio built with React, Tailwind CSS and Framer Motion.

Getting started

1. Install dependencies

```bash
npm install
```

2. Run dev server

```bash
npm run dev
```

Build

```bash
npm run build
```

Notes

- This project uses Vite + React. Tailwind needs PostCSS (configured).
- Replace contact details and resume link in `src/components/Contact.jsx` and `src/components/Hero.jsx`.
